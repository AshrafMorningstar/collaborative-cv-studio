## :floppy_disk: Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AshrafMorningstar/collaborative-cv-studio.git
   cd collaborative-cv-studio
   ```

2. **Install dependencies:**
    npm install

3. **Run the application:**
    npm start
